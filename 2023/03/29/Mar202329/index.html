Gaussian Blur



Here is an example how blurred image may look like:



This blurred image is based on the much more detailed one (Credits: NASA):



Gaussian Blur filter is used for remove high frequency components from image. This is important before the edge detection operation - details on image may produce false positive edges. Gaussian blur may be used also for resizing image.

Gaussian blur algorithm has two stages. In the stage one Gaussian kernel is created. In the stage two image is convolved by created kernel.

Two dimensional Gaussian kernel is created due to the equation:


The full algorithm is more complicated and it is explained here. I will write only necessary information below algorithm implementation. The core of the algorithm is a kernel:

The first line of code creates square matrix of zeros of size (ksize x ksize). If user pass as an argument number 3 then matrix will be look like:

[0 0 0
 0 0 0
 0 0 0]

In the next step we are creating values for matrix (these values depends on the matrix size). In our example variable w will be equal to (-1).

Inside the for loop our matrix is filled with the proper values. After ending of loop it will be:

[-1 0 1
 -1 0 1
 -1 0 1]

And this is the first matrix (x in the equation provided at the beginning). We must create also other, transposed one and for this we are using the last line of code (y in the equation provided at the beginning). And our transposed matrix:

[-1 -1 -1
  0   0  0
  1   1  1]

#2 Convolution with the image

Normalized kernel is returned to the basic function. Operation similar to showed in the previous note is realised. Image is convolved with the kernel and new image (blurred) is returned.

function blurred_image = gaussianBlur(image, sigma, kernel_size)
    
    // Checking kernel_size
    

<span style="color: #008800; font-weight: bold;">if</span> <span style="color: #007020;">round</span>(kernel_size<span style="color: #333333;">/</span><span style="color: #0000dd; font-weight: bold;">2</span>) <span style="color: #333333;">==</span> (kernel_size<span style="color: #333333;">/</span><span style="color: #0000dd; font-weight: bold;">2</span>) <span style="color: #007020;">then</span>
    
    <span style="color: #007020;">error</span>(<span style="background-color: #fff0f0;">'Kernel Size must be odd number'</span>)
    <span style="color: #007020;">abort</span>
    
<span style="color: #008800; font-weight: bold;">else</span>
    
    gaussKernel = createGaussianKernel(sigma, kernel_size)
    blurred_image = <span style="color: #007020;">conv2</span>(image, gaussKernel, <span style="background-color: #fff0f0;">"same"</span>)
    
<span style="color: #008800; font-weight: bold;">end</span>



endfunction
